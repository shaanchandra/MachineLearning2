## LAB Assignments
### Lab 1: Independent Component Analysis
#### Topics:
- Independent Component Analysis (ICA) on mixtures of audio files

### Lab 2: Inference in Graphical Models
#### Topics:
- The sum-product algorithm
- The max-sum algorithm
- Medical Graph
